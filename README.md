# 🦟 DengueX: Dengue Monitoring Platform

# 📌 Overview

DengueX is a web-based platform for dengue awareness, monitoring, and data visualization. It combines AI-based modules with structured public health data to present information in a clear and accessible format.

This project was developed as a Final Year Project (FYP) with a focus on practical implementation, data clarity, and system reliability. It is intended for academic and research use only.

# 🧩 Core Modules

# 🤖 1. Dengue Awareness Chatbot

A rule-based chatbot that provides predefined, reliable information on:

-Symptoms
-Transmission
-Prevention

The system is restricted to dengue-related queries to ensure consistency.

# 🦟 2. Mosquito Detection Module

A computer vision-based classification system that distinguishes between:

-Aedes mosquitoes
-Non-Aedes mosquitoes

The model is designed for stable and generalizable predictions, not maximum accuracy.

# 📊 3. Analytics Dashboard

A visualization module built on publicly available dengue datasets, providing:

-Province-level case distribution
-City-level trends
-Seasonal patterns
-Year-wise comparisons

This module focuses on descriptive analysis, not prediction.

# 🛡️ 4. Prevention & Awareness

Provides structured guidance including:

-Preventive measures
-Seasonal precautions
-Basic symptom awareness

# 🧑‍💼 5. Admin Panel

-Dataset updates
-Dashboard management
-Controlled content visibility

# ⚙️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Python (Django), Fast APIs
Database: SQLite
AI: Rule-based NLP, Deep Learning (CNN, EfficientNet, YOLO,Mobilenet,Kaggle)
Visualization: Interactive chart-based dashboards
